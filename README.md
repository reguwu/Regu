## Overview
This is a software engineer portfolio website that I built to showcase stuff.

## Development

Create a `.env.local` file in the root directory

```bash
RESEND_API_KEY=
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

## Tech Stack

- Next.js
- CSS Modules
- MDX
- Radix UI
- Framer Motion
- React Email + Resend
- Yet Another React Lightbox
