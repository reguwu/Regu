## Overview
Uneven Pool is a 3D Third Person Control PC Game where the goal is to use one ball to knock another ball into a hole.
It is a combination of Pool and Golf.

## Responsibility

- Designing/Implementing the game UI/UX, such as game menu, buttons, sound effect, etc...
- Creating the 3D models for each level of the game
- Programming the physics of interactable in-game objects
